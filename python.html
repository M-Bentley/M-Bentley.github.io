An Itchy & Scracthy/Space Invaders inspired game.

The player is able to fire chainsaws to prevent the cats from reaching the ground.

If the enemies are able to reach the ground the game is over.

self.image = pygame.image.load("H:\Itchy.png") self.image = pygame.transform.scale(self.image, (50,100))

These lines are what we used to load the image into the game as well as scale the photo's size.
